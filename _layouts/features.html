How do I get data?

Using our desktop app you can turn any website into a table of data or an API in just a few minutes without writing any code.

{% include appDownload.html %}
 

There are three methods of data extraction:

Extractors

Extractors are the most straightforward method of getting data, they allow you to turn an unstructured web page into a table of data.

Crawlers

A crawler is an extractor with legs; it allows you to go to every page of a website and extract all of the data on every page that matches the pattern you define.

Connectors

A connector is an extractor that lets you interact with a web site, for example via a search field, to get to your data.

What can I do with my data?

Create

Datasets allow our users to mash together live and static data from in any combination from Connectors, Extractors, Crawlers and Mixes.

Share

Generate a unique link to your Data Set to share it with anyone

Export

Download your Data Set as CSV, HTML, or XLS

Integrate

Integrating our APIs is easy, whether it's into Excel or iOS. We make it simple to get data into just about anything!

Mix

Search across multiple sites by mixing together multiple APIs.

Store

All your data is stored on our cloud servers so you can access data from anywhere!
